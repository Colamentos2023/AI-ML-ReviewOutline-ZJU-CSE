## 一、选择题（2 分 × 20）

1. 下列哪一项**不**属于人工智能的典型应用？ *[补全]*
2. 下列哪个算法是**按照某种规则遍历搜索空间**的算法？
   - A. DFS  B. BFS  C. A\*  D. 优先级搜索（best-first）
3. 机器学习是：
   - A. 人工智能的一个分支
   - B. 数据分析的一种技术
   - C. 以上都是 *[补全]*
   - D. 以上都不是
4. 下列哪种算法**不是**盲目搜索？ *[补全]*（候选：DFS / BFS / 一致代价 / A\*）
5. 欠拟合（underfitting）的定义是？
6. 下列哪些是常用的**算法评价指标**？（多选）
   - A. 准确率、召回率  B. MSE  C. MAE  D. AUC-ROC 曲线
7. 下列哪些指标可以评价**回归**任务？（多选）
   - A. MAE  B. MSE  C. binary cross-entropy  D. Smooth L1
8. **贝叶斯计算**：给定先验与似然，计算后验概率。 *[补全]*
9. 下列哪种算法属于**强化学习**？ *[补全]*（候选：Q-learning / SVM / KNN / K-means）
10. 下列关于**人类智能**的定义说法正确的是？ *[补全]*
11. 若**假设空间很大**，最适合使用的搜索方法是？ *[补全]*（启发式搜索 / 穷举 / …）
12. **人脸识别**算法的实现顺序是？（含特征点匹配等步骤） *[补全]*
13. 若存在最优解，下列哪个算法**一定能找到**？
    - A. BFS  B. 启发式搜索  C. DFS  D. 有限深度搜索
14. 下列哪个子句集**不可满足**？
    - A. $\{\ldots,\lnot p, p\}$
    - B. $\{p\lor q,\ \lnot p\lor q,\ p\lor \lnot q,\ \lnot p\lor\lnot q\}$
    - C. … *[补全]*
    - D. … *[补全]*
15. **概念学习**的定义是？ *[补全]*

## 二、填空题（1 分 × 15）

1. A\* 算法中 $g(n)$ 称作 ________（实际代价/已付出代价），$h(n)$ 称作 ________（启发函数/估计代价）。
2. 人工智能的短期目标是 ________，终极目标是 ________。
3. 人工智能的三大学派是 ________、________、________（符号主义 / 连接主义 / 行为主义）。
4. BFS 存储**待搜索**节点的数据结构是 ________（队列），DFS 存储**已搜索**节点的数据结构是 ________（栈）。
5. 强化学习中，有信息的情形叫 ________，无信息的情形叫 ________。
6. Find-S 用 ________ 序的方法，在 ________ 结构上实现，每一步得到的假设都是在该点上与训练样例一致的 ________ 假设。

## 三、大题

1. **对抗搜索与 $\alpha$-$\beta$ 剪枝**：
   1. 给定博弈树，求各节点的倒推值；
   2. 简述 $\alpha$-$\beta$ 剪枝的基本原理；
   3. 给出剪枝结果并说明剪去的分支。
2. **一阶逻辑归结证明**：已知"凡是干净的东西就有人喜欢"和"没人喜欢苍蝇"。
   1. 定义谓词；
   2. 写出逻辑表达式；
   3. 转换为标准子句集；
   4. 用归结反演证明"苍蝇不干净"。
3. **开放题**：过程控制流程包含"数据收集与分析 → 系统建模 → 控制系统设计 → 离线验证与测试 → 实地部署"。请设计一套流程，结合本门课程所学至少 **三种** 方法说明其应用及可行性。
4. **决策树**是什么？其优缺点是什么？
5. **朴素贝叶斯**的原理是什么？典型应用场景有哪些？
6. 谈谈你对**梯度消失**的理解。
7. **Q-learning** 的基本原理是什么？在强化学习中它用于解决什么问题？
